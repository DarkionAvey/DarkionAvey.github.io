Corsair Link Windows 8/8.1 compatibility fix

Corsair Link software seems to be broken after Windows 8.1 update. Fortunately, there is a fix.

  1. Hit the start menu, and type "regedit" (Registry editor). Right-click it, then choose "run as administrator" to gain the needed privileges.

  2. From the left side folder tree, navigate to the following key:

    HKLM\SYSTEM\CurrentControlSet\Enum\USB\VID_1B1C&PID_0C04

  3. Navigate to "Device Parameters"

  4. Change the following DWORD values from 1 to 0

      AllowIdleIrpInD3
DeviceSelectiveSuspended
EnhancedPowerManagementEnabled
SelectiveSuspendEnabled

  5. Do this for every sub folder inside "VID_1B1C&PID_0C04". In my case, it’s just one folder.

  6. Restart your machine

After restarting, you will notice that the missing LED controls and fan controls are visible again.
