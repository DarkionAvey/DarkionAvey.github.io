Need for Speed Rivals 60 FPS mod

Darkion Avey

Aesthetics and animations

  1. Locate the game NFS14.exe file. Usually, it’s located in your Origin library (C:\Program Files (x86)\Origin Games\Need for Speed™ Rivals\)

  2. Create a shortcut of the exe file by right-clicking it > send to > desktop (create a shortcut)

  3. Right click on the shortcut, and then click on properties.

  4. In the target field, add the following code at the end (after the " quotation mark)

    -GameTime.MaxSimFps 60 -GameTime.ForceSimRate 60+

  5. Launch the game from the shortcut
