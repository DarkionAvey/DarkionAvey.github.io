How to use TexMod on Windows PC

Darkion Avey

Aesthetics and animations

TexMod is an old tool made to replace the textures of any Dx9 game without modifying the original files of the game. Follow the steps below to get going:

  1. Download TexMod from other sources (file is no longer hosted on this domain)

  2. Launch the application. Note: anti-virus applications might label TexMod as a virus, which is not. This happens because TexMod modifies DirectX files during runtime just like a virus would modify the system.

  3. Click on the yellow folder icon, and select your game exe file. Note: some games (such as Mass Effect 2, The Witcher 2 and most games released early 2010’s) have a launcher application, you have to select the game exe file and not the launcher exe .

  4. Click on the small folder icon to browse and select your TexMod files (TPF files).

  5. Hit Run and enjoy your new textures.
